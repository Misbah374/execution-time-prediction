# 🚀 Execution Time Prediction using Polynomial Regression

## 📌 Overview
This project focuses on predicting the execution time of a program based on input parameters using **Polynomial Regression implemented from scratch (without scikit-learn)**.

The key objective is to understand how **feature engineering** (especially interaction terms) impacts model performance.

---

## 🎯 Problem Statement
Given:
- `input_size (n)`
- `loop_iterations (m)`

Predict:
- `execution_time`
